# Voice Assistant (Python)

A simple Python-based voice assistant that listens to user commands through a microphone,
converts speech to text, processes the request, and responds using text-to-speech.

## Features
- Voice input using microphone
- Text-to-speech response
- Greeting the user
- Telling current time
- Opening websites using voice commands

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3

## How to Run
1. Clone the repository:
   git clone https://github.com/gaurav-eng/voice_assistant_p1.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the program:
   python voice_assistant_p1.py
